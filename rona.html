A 2D video game where the player controlls an alligator that tries to scare students into social distancing. Players get points for scaring and causing students to scatter.


This project introduced me to programming with C#, using the Unity game engine and using Photoshop for sprites and backgrounds. I created all the art for this game and most of the programming behind the scenes.
