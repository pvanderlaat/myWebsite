Spotify Song Linker

Python | Spotify API

This is a project I created for my Data Structures and Algorithms class where we had to use an existing dataset of at least 10,000 data points, store them in a graph, and traverse the graph using different algorithms. My team and I used Spotify's API to gather our data points. My role in the project was to set up the Spotify API code and hook it up to the graph creation process. I also played a major part in implementing the dijkstra's algorithm part.
