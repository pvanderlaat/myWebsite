Clubfinity

JavaScript - ReactNative - MongoDB - Node.js - Figma - GitLab

UF has hundreds of different student organizations and they all use different tools to manage their activity such as Facebook, slack, discord, instagram and more. Clubfinity is an app being created by the UF Software Engineering Club that hopes to reduce all this confusion. The goal of this project is to connect all the clubs on campus into one central location.


As a UX designer and technical officer, my main focus is the front-end design of the app, but I also help develop the code for the backend and frontend.
