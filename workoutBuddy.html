Workout Buddy

MERN STACK

This personal project of mine is a web app that helps me keep track of my workouts and which workouts I should do to maximize my recovery/work times. I basically created a MERN app completely from scratch. For the backend, I created an API from scratch that allowed for GET, DELETE, POST, and PATCH functionality. For the frontend, I used a REACT template to nicely display this workout information.


I deployed this project through AWS's EC2 service.
